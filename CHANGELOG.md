## [1.0.0] - 2026-04-01

### Added

- **Annotation Tools**
  - Highlight text with multiple color options (yellow, green, blue, pink, orange)
  - Underline text annotations
  - Strikethrough text annotations
  - Sticky notes (expandable/collapsible comments on pages)
  - Drawing/freehand tool with color and thickness options
  - Eraser tool for removing drawings
  - Annotation persistence (all annotations saved independently from PDF)

- **Bookmark Management**
  - Add/remove bookmarks with Cmd+B
  - Named bookmarks with custom titles
  - Bookmark list sidebar
  - Click-to-navigate bookmark functionality

- **Full-Text Search**
  - Instant search across entire documents
  - Highlighted search results
  - Navigate between matches with keyboard shortcuts
  - Case-insensitive search by default

- **Outline Navigation**
  - Hierarchical document outline display
  - Multi-level outline support (up to 5 levels)
  - Collapsible/expandable outline sections
  - Click-to-navigate outline items

- **Thumbnail Sidebar**
  - Visual page previews
  - Grid display of all pages
  - Current page highlight
  - Click-to-navigate thumbnails

- **Recent Files**
  - Quick access to recently opened documents
  - Display of filename, path, and last opened time
  - Automatic cleanup of non-existent files
  - Persistent history across sessions
