## 1. 项目概述

PDF-Ve v2.0 是 v1.0 的功能增强版本，在 v1.0 基础上新增以下功能：

**v2.0 新增功能**：
1. **页面导航Fallback**：当 PDF 无目录时，侧栏显示"第1页/第2页..."文字导航
2. **文件路径显示**：菜单栏实时显示当前打开文件的路径
3. **快速打开下拉菜单**：工具栏右侧增加下拉按钮，提供打开/打开网络文件/最近文件快速访问
4. **窗口状态记忆**：记住窗口大小和侧边栏标签页，下次打开自动恢复

**技术栈**（同 v1.0）：
- 语言：Swift 5.9+
- UI 框架：SwiftUI + AppKit（NSViewRepresentable）
- PDF 引擎：PDFKit（Apple 系统框架）
- 数据持久化：SwiftData（macOS 14+）+ UserDefaults（窗口状态）
- 构建工具：Xcode 15+ / Swift Package Manager
- 最低系统：macOS 13 Ventura

## 5. 开发规范

### 5.1 代码规范（同 v1.0）
- Swift 官方风格指南
- `@MainActor` 修饰所有 ViewModel
- 异步操作使用 `async/await`

### 5.2 提交规范
```
feat(v2): add page navigation fallback
feat(v2): add window state memory
feat(v2): add quick open menu
```

### 5.3 任务约束
- **代码变更**：≤ 200 行
- **涉及文件**：≤ 5 个
- **执行时间**：≤ 30 分钟

### 5.4 项目特定验证规范

#### SwiftUI macOS 布局验证清单

**侧栏导航验证**:
- [ ] 无目录时显示"第X页"文字列表
- [ ] 有目录时显示原有目录结构
- [ ] 页面导航与其他 Tab 切换正常

**窗口状态验证**:
- [ ] 窗口大小在关闭时保存
- [ ] 窗口大小在重新打开时恢复
- [ ] 侧边栏标签页在切换时保存
- [ ] 侧边栏标签页在重新打开时恢复

**快速打开菜单验证**:
- [ ] 下拉菜单正确展开
- [ ] 打开文件功能正常
- [ ] 打开网络文件功能正常
- [ ] 最近文件列表正确显示

#### 验收标准三级验证格式

所有 View 层任务必须使用以下验收标准格式：

```markdown
**验收标准**:
- [ ] **数据展示验证**: [静态渲染正确，空状态处理等]
- [ ] **布局验证**: [尺寸符合 UI-Layout-XXX、不挤占核心内容区等]
- [ ] **交互验证**: [事件触发验证、状态反馈验证、数据流转验证等]
```
