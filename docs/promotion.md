## Reddit (r/macapps)

### English
**Title: Lumen - A free, open-source PDF reader built with SwiftUI (just released on GitHub)**

Hey r/macapps,

I've been frustrated with PDF readers that charge a subscription just to export annotations or sync bookmarks. So I built Lumen.

**What it does:**
- Annotations (highlight, underline, notes)
- Bookmarks with custom naming
- Full-text search with highlighted results
- Outline view for navigation
- Thumbnail sidebar

**Why it's different:**
- 100% free, no paywalled features
- Open source (MIT license)
- Built natively with SwiftUI — respects macOS conventions
- No account, no cloud, no data collection

As a PM who codes on the side, I know how hard it is to find a PDF reader that just... works without locking features behind a paywall. This is my attempt at the tool I always wanted.
